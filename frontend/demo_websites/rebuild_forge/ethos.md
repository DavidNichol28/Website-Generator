# Rebuild Forge's Platform
---
## Core Values

### Freeness
	- Lowest possible resource costs for userbase
	- Non-restrictive use
### Openness
	- Made to be used with other people in mind
	- Zero possible restrictions to any code
### Accessiblity
	- Projects always have the lay-user front-and-center
	- Projects are built using the Foundation's designated toolkit to remove code-nerd-gatekeeping
### Transparency
	- No fancy bullshit
### Usefulness
	- Function over form
	- Never leave something unclear, confusing, or buggy
### Modularity
	- Keep things from being deeply connected - even when its a serious pain
